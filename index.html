권한 확인 & 주인변경

👤 매장 플레이스를 관리하고 있는 네이버 ID 확인

아래와 같은 상황에서 주인권한 확인이 필요합니다:

  • 사장님이 플레이스를 관리하지 않고 있다고 하시는 경우
  • 스마트플레이스를 가입할 때 대표자가 아닌 직원명의로 가입한 경우
  • 사장님이 플레이스를 관리하는 네이버ID를 모른다고 하시는 경우

🔍 확인 방법

  1. 네이버에서 '스마트플레이스' 검색 후 접속 → https://new.smartplace.naver.com/
  2. 로그인 후 '내 업체'에 내 매장 플레이스가 노출되어있는지 확인
  3. 내 매장 플레이스가 보이지 않는다면 → 아래 주인변경 진행
🔄 주인변경 진행 (필요한 경우만)
❶ 스마트플레이스 접속 후 관리할 네이버 ID로 로그인 → [업체찾기] 클릭 ▼
  1. https://new.smartplace.naver.com/ 접속
  2. 앞으로 매장을 관리할 네이버 ID로 로그인
  3. [업체 찾기] 클릭 후 전화번호 / 사업자등록번호 / 업체명 중 하나로 조회
❷ 조회 결과에서 내 매장 선택 후 [주인변경] → ARS 인증 ▼
  1. 내 매장을 찾아 [주인변경] 클릭
  2. [ARS로 주인변경] 클릭 → 매장 대표 전화번호로 ARS 인증
  3. ARS 인증 불가 시 → [서류로 주인 변경]으로 사업자등록증 제출 (3~5 영업일 소요)
✅
인증 완료 시 즉시 주인 변경 완료. 해당 ID로 스마트플레이스 로그인 후 '내 업체'에서 확인 가능합니다.
2

예약 서비스 가입

📝 네이버 예약 서비스 가입 (3단계)
❶ 스마트플레이스 접속 후 [예약/주문] 버튼 클릭 ▼
  1. 네이버에서 '스마트플레이스' 검색 후 접속
  2. 플레이스 관리중인 네이버 ID로 로그인 후 [예약/주문] 버튼 클릭
❷ 예약/주문 서비스 추가 클릭 ▼
  1. [예약주문] 상단 탭 클릭
  2. [예약/주문 서비스 추가] 버튼 클릭
❸ 정보 입력 ▼

아래 정보를 정확히 입력합니다:

  • 서비스명: '업체명'으로 입력
  • 업체사진: 매장 관련 사진 (내부사진 등)으로 등록
  • 배경색상: 선택
  • 대표자명: 사업자등록증 상 대표자명 입력
  • 제공하는 항목: 매장에 해당하는 항목 선택 (시설정보, 서비스정보)
  • 연락처: 예약문의, 이메일, 관리자 연락처 (필수입력 O 항목 확인)
  • 업체서류: 필수 입력 X
❹ 예약 유형 선택 및 완료 ▼

예약 유형 선택:

  • 시간까지 선택 클릭
  • 유형: 일반형 클릭 (날짜와 시간을 예약, 일반형만 네이버 주문이 가능)
⚠️
예약유형은 선택 후 변경할 수 없습니다. 실제 운영에 가장 적합한 유형으로 선택해 주세요.

서비스 등록이 완료되면 이어서 [예약 상품 등록하기]를 진행합니다.

3

예약 상품 등록 & 페이 가입

📦 네이버 예약 상품 등록하기
❶ 예약 상품 유형 선택, 상품명 및 상품 소개 입력 ▼
  • 유형: 일반 예약, 룸/좌석 예약, 포장 예약, 기타 예약 중 선택
  • 방문시 앉을 수 있는 자리: 홀, 룸, 바(카운터), 항가, 테라스, 루프탑(옥상), 야외, 단체석, 연인석 중 선택
  • 예약 상품명: 업체명 또는 상품명 입력 (필수)
  • 예약 상품 소개: 매장 소개 작성 (예: "그린식당은 100% 예약제로 운영합니다.")
  • 사진 추가
❷ 예약 방문 관련 유의 및 안내 사항 작성 ▼
  • 예약 꼭 확인해야할 사항: 예약자 요청 사항 입력
  • 룸 이용 안내: 필수 입력 아님 (예: "4인 이상 이용 가능합니다.")
  • 이용안내: 필수 입력 아님 (노키즈존, 반려동물 가능 여부 등)
  • 이용시간 안내: 필수 입력 아님 (예: "라스트 오더는 20:30 입니다.")
💡
고객이 예약 시 확인하는 중요 정보이므로, 꼼꼼하게 작성하면 예약 전환율이 높아집니다.
❸ 예약 운영 시간 설정 ▼
  • 항상 운영 계획: "네, 늘 운영합니다." 또는 "특정 기간만 운영해요!"
  • 예약 받는 시간 간격: 30분 단위 / 1시간 단위
  • 예약 운영 시간: 모든 영업일 같아요 / 평일·주말 달라요 / 요일별로 달라요
  • 운영 시간: 시작 ~ 마지막 시간 (고객이 선택할 수 있는 마지막 시간)
  • 휴무일: 정기 휴무일, 공휴일 중 휴무일 설정
❹ 예약 상품 인원 설정 ▼
  • 예약 관리 기준: 인원으로 관리 / 예약건수로 관리 중 선택
  • 하루 예약 가능 인원(명): 총 인원 설정
  • 예약 방식: 인원수 받기, 예약자 구분하기, 금액까지 받기 중 선택
  • 한 번에 예약 가능한 인원: 최소 ~ 최대 인원 설정
💳 네이버페이 가입 (예약금/선결제 받을 매장만 해당)
💡
예약금/선결제 없이 예약 받을 매장인 경우, 이 단계를 건너뛰고 STEP 4 검수 신청으로 이동합니다.
❶ [N Pay 결제 사용하기] 클릭 ▼

예약상품 화면에서 N Pay 결제 사용하기를 ON으로 설정합니다. 네이버페이에 가입되어 있지 않으면 가입 안내 팝업이 노출됩니다.

❷ N페이 결제사용 클릭 > [네이버페이 사용하기] 버튼 클릭 ▼

스마트플레이스 > 솔루션 찾기 > 판매/결제 > "예약, 주문시 N페이 결제사용" > 자세히 > [네이버페이 사용하기] 버튼 클릭

❸ 페이 가입 진행 ▼
  1. 약관 전체동의 후 동의 클릭
  2. 사업자등록증 사진 올리기
  3. 사업자 정보 입력 (사업자 유형, 등록번호)
  4. 정산 정보 입력 (은행, 계좌번호, 대표자명)
⚠️
사업자등록증 올릴 때 반드시 마스킹(보이지 않게) 처리 필요!
1) 주민번호 뒤 7자리   2) 전화번호   3) 이메일 주소
★
사업자등록증 상 대표자명과 계좌명(계좌주인)이 반드시 일치해야 합니다.

검수 신청 & 예약 노출

🔎 검수 신청
❶ 예약 상품 노출 ON ▼

예약상품 화면에서 노출중을 ON으로 변경합니다. 예약 신청과 동시에 확정 설정도 확인하세요.

❷ 예약주문 > 설정 > 운영설정 > 검수신청 진행 ▼

스마트플레이스에서 예약주문 > 설정 > 운영설정으로 이동 후 [검수신청] 버튼을 클릭합니다.

검수가 완료되면 메일로 결과를 안내 드립니다. 검수 완료 후 예약받기 ON으로 변경하세요.

⚡ 검수 신청 후 패스트 트랙

검수 신청 후 대행사 진행 필요 사항:

대행사 검수신청 후 매장명 및 매장 플레이스 링크를 구글 시트에 업데이트

네이버 일반 검수 (영업일 기준)

  • 당일 낮 12시 이전 신청 → 당일 19시까지 검수 완료
  • 당일 낮 12시 이후 신청 → 익일 오후까지 검수 완료

긴급 검수: 카카오톡 단체방에 요청 (즉시 검수가 필요한 긴급 건에만 제한적으로 대응)

구글 시트 리스트의 매장 검수 후 검수 결과 대행사에 이메일로 전달 (1일 1회)

대행사 아래 경우에 따라 사업주 대상 안내

  • 검수 통과 시: 예약 버튼 노출 방법 안내 진행
  • 검수 보류 시: 보류 사유 확인하여 수정 후 재검수 요청 (신규 일자로)
⚠️ 검수 보류 시 수정 방법
  1. 스마트플레이스 [예약주문 > 설정 > 운영설정] 버튼을 클릭
  2. 상세보류 사유 확인 > 수정 > [재검수신청] 클릭
💬
보류된 업체와 사유는 이메일로 1일 1회 전달 드릴 예정입니다.
* 참고: 검수결과는 플레이스 관리중인 네이버 아이디로도 메일이 발송됩니다.
👀 검수 통과 후 예약 버튼 노출 및 운영 점검

검수 완료 후 아래 3가지를 반드시 확인하세요:

❶ 예약/주문받기 ON

[예약주문 > 설정 > 운영설정]에서 예약/주문받기 ON으로 변경, 플레이스 연결하기

❷ 예약 상품 ON 처리

[예약주문 > 예약 > 예약상품]에서 노출중 ON으로 변경

❸ '예약신청과 동시에 확정' 설정

캐치테이블 2차연동 요청

🔗 캐치테이블에 2차연동 요청 (캐치테이블 사용 식당 한정)
캐치테이블 2차 연동이란?
좌석 재고의 마스터가 캐치테이블이 되며, 캐치 어드민에서 네이버예약의 예약을 모두 관리할 수 있음. 즉, 식당에서는 네이버예약 페이지를 직접 관리할 필요가 없습니다.
❶ 캐치테이블에서 2차연동 서비스 신청 ▼
  1. [캐치테이블 서비스 사용 가이드 > 연동신청하기]로 이동
  2. 연동 신청 후 신청화면을 캡처해 주세요.
  3. 제출 후 캐치테이블 채팅 상담으로 말씀해 주세요.
  4. 연동 작업은 순차적으로 진행되며, 최대 1~2일 정도 소요됩니다.
❷ 예약 상품 ON 처리 및 연동 결과 확인 ▼

연동 완료 후 예약 상품을 ON 처리합니다.

⚠️
캐치테이블에서 2차연동 거절하면:
캐치테이블에 거절 사유를 문의하고, 식당에서 직접 네이버예약을 사용할 수 있도록 설득 및 교육 진행
